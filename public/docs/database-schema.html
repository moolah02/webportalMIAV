Key Relationships Summary

FromRelationshipTo
employeesbelongs todepartments, roles
pos_terminalsbelongs toclients
job_assignmentsbelongs toemployees (technician), clients, regions, projects
visitsbelongs toemployees; linked to job_assignments
visit_terminalsbelongs tovisits; references pos_terminals
ticketsbelongs toemployees (tech + assignee), pos_terminals, clients
ticket_stepsbelongs totickets, employees
asset_requestsbelongs toemployees; has many asset_request_items
asset_request_itemsbelongs toasset_requests, assets
project_terminalspivot betweenprojects, pos_terminals
