Purpose

The MIAV system is a comprehensive field service management platform designed to manage POS terminal deployments across multiple regions. It enables administrators to assign jobs to field technicians, track terminal servicing, manage assets, and handle support tickets — all in real time via a web dashboard and mobile application.

3. System Modules & Features

3.1 Dashboard & Analytics

  • Role-specific dashboards (admin, manager, technician)
  • Real-time statistics: terminals, jobs, tickets, technicians
  • Regional performance charts (ApexCharts)
  • System health alerts

3.2 POS Terminal Management

  • Full CRUD for terminals with 30+ data fields
  • Bulk CSV/Excel import with column mapping
  • Export to CSV/Excel/PDF
  • Terminal status tracking (active, offline, maintenance, faulty, decommissioned)
  • Service history, notes, and scheduling
  • GPS coordinates with map integration support
  • Per-terminal analytics and statistics

3.3 Job Assignment Management

  • Create and assign field service jobs to technicians
  • Schedule jobs by region and date
  • Job lifecycle: assigned → in_progress → completed/cancelled
  • Check-in/check-out with location tracking
  • Photo and note documentation per job

3.4 Site Visit Reporting

  • Mobile-submitted visit reports with evidence photos
  • Terminal condition and status snapshot per visit
  • Customer signature capture
  • Offline-first: visits saved locally and synced on reconnect
  • Action points and follow-up tracking

3.5 Ticketing & Support

  • Create tickets for POS terminal issues or internal issues
  • Priority levels: low, medium, high, urgent
  • Public tickets (any technician can attend) vs Direct (assigned to specific technician)
  • Staged resolution system: multi-step workflow with transfers between employees
  • Comment thread per ticket
  • Full status audit trail
  • Status flow: open → in_progress / pending → resolved / closed / cancelled

3.6 Asset Management

  • Asset inventory with categories, stock levels, and pricing
  • Low-stock alerts based on configurable minimum levels
  • Asset request workflow: submit → approved/rejected → fulfilled
  • Asset assignment to employees with return tracking
  • Same-day request support

3.7 Client Management

  • Full client profiles with contacts, contracts, and status
  • Linked terminals and projects per client
  • Client statistics and activity history

3.8 Reporting

  • Pre-built reports: assignments, terminal performance, technician productivity
  • Custom report builder with drag-and-drop field selection
  • Save and share report templates
  • Export to PDF, Excel, and CSV

3.9 Employee & Access Management

  • Employee profiles with roles, departments, and permissions
  • Granular permission system (50+ individual permissions)
  • Role management with permission bundles
  • Account activation/deactivation

First Login Steps

  1. Navigate to the web dashboard URL

  4. Assign roles and permissions per user from Admin → Roles
  5. Mobile app users will use the same email/password credentials via the app
